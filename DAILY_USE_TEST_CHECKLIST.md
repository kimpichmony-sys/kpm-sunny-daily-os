# Daily Use Test Checklist

Use this checklist after deploying the live KPM Sunny Daily OS app.

- [ ] Open live Vercel link.
- [ ] Confirm app loads.
- [ ] Confirm Today Command screen appears.
- [ ] Confirm Current Mission is clear.
- [ ] Press Start Mission.
- [ ] Confirm timer preset matches task type.
- [ ] Mark task done.
- [ ] Confirm next mission updates.
- [ ] Check Nanno's Sleep Boost appears in Night Shutdown.
- [ ] Check Settings version/changelog.
- [ ] Export backup data.
- [ ] Refresh page and confirm data stays.
- [ ] Test on iPhone.
- [ ] Test on Windows Chrome.
- [ ] Note anything annoying or confusing.
